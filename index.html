Developer Information (For Power Automate Setup)

The lead data is now saved to Firestore. To complete the WhatsApp integration, you must configure a secure backend (like **Power Automate**) to pull this data and call the actual WhatsApp Cloud API.

The **API Call Payload** you should use in your Power Automate HTTP action looks like this (you'll use the data saved in Firestore): 

{
  "messaging_product": "whatsapp",
  "to": "[[Customer Phone Number from Firestore]]",
  "type": "text",
  "text": {
    "body": "Hi [[Customer Name]], thanks for your inquiry about Ampere EVs! We received your message: '[[Customer Query]]'. A sales representative will contact you shortly."
  }
}

**Endpoint:** `https://graph.facebook.com/v19.0/[YOUR_PHONE_NUMBER_ID]/messages`
**Authorization:** `Bearer [YOUR_SECURE_API_TOKEN]` (This token must only be stored in Power Automate/backend.)
